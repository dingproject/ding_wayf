This module adds support for Where Are You From (WAYF) services to the ding
project. It is a requirement that you have a contract with WAYF (http://www.wayf.dk/)
about nemlog-in.

This module requires that you install SimpleSAMLphp, which is described in 
details at http://ting.dk/wiki/installation-af-nemlog-gennem-wayf. Remember to
set permissions for the module, so users are allow to use WAYF.
